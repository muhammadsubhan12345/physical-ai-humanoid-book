## What Is an AI Agent?

An AI agent is a system that:
- Observes the environment
- Makes decisions
- Takes actions to achieve goals

In robots, agents operate continuously.

---

## Agent Architecture in Robots

A typical robot agent includes:
- Perception module
- Planning module
- Control module
- Learning module

These modules communicate in real time.

---

## Why Agents Are Important

Agents allow robots to:
- Adapt to new situations
- React to uncertainty
- Operate autonomously
- Learn from experience

Without agents, robots would be simple machines.

---

## Physical AI and Agents

Physical AI systems rely on agents to:
- Interpret sensor data
- Decide safe actions
- Control physical movement

Agents are the intelligence of humanoid robots.

---

## Key Takeaways

- Robots use AI agents for autonomy
- Agents connect perception, planning, and action
- Physical AI depends on intelligent agents
